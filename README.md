# Real-Time Financial Fraud Analytics Platform

## Business Problem
Financial systems process millions of transactions per day. Fraud teams need
real-time signals to detect suspicious activity while analytics teams require
clean historical data for reporting and risk analysis.

This project simulates real-time financial transactions, detects fraud patterns
using streaming rules, and delivers analytics-ready data and dashboards.

## Architecture (High Level)
Synthetic Transaction Generator → Kafka → Fraud Processing → BigQuery → Dashboard

## Tech Stack
- Python
- Apache Kafka
- BigQuery
- SQL
- Looker Studio (Dashboard)
